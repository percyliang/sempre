Instructions

Write trivia questions using the information from the highlighted table (green borders).

Each question must follow the question template. You can use words with the same meaning (e.g. for the pattern "... first/last ..." you can use "latest" as well). You can also randomize the question template by clicking the button.

Please use different grammatical constructs. See the examples for some ideas. However, a person must be able to answer the questions using only the texts from the highlighted table.

Examples:   (http://en.wikipedia.org/wiki/List_of_Presidents_of_the_United_States#List_of_presidents)

Good Questions

  • presidents who were also vice presidents
  • How old was Lincoln when he took office?
  • Which party has had the most number of presidents?
  • How long did Lincoln live?
  • Which presidents were under 50 when they took office?
  • the average age of the first ten presidents when they first took office

Bad Questions

  • Which presidents were born in New York? (requires outside information)
  • presidents with glasses (requires images)

If you have any problems or suggestions, please contact us.
